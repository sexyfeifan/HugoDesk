# 更新说明（Changelog）

## v0.3.9 - 2026-03-04

### 版本目标 🎯

本版本聚焦于“减少过度设置，固定 Hugo 正确发布路径”，让 HugoDesk 更稳定地替代命令行发布博客。

### 主要调整 ✅

- 固定发布策略为 GitHub Actions（移除易误导的发布分支策略切换）。
- 移除“推送时排除 `hugo.toml`”选项，确保 Hugo 关键配置始终参与发布。
- 精简发布控制台功能，只保留核心流程入口。
- 精简项目页快捷操作与项目配置包操作，减少重复按钮。
- 保留并强化“一键生成 Pages Workflow”能力，用于补齐 GitHub Pages 部署链路。
- 清理无入口旧逻辑，降低后续维护复杂度。

### 发布流程变化 📦

发布页主流程调整为：

1. 同步远程
2. 一键检测推送与部署
3. 提交并推送

### 兼容与安全 🔒

- `.hugodesk.local.json` 仍作为项目本地配置包使用，不进入 Git。
- Token 仍通过安全机制参与发布，不在日志中明文泄露。

### 历史版本

- v0.3.8：引入 Pages Workflow 一键生成与部署链路检测。
- v0.3.7：修复目录定位、编辑器工具和预览图片路径等关键问题。
